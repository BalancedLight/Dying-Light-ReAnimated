# Third-party assets

DL ReAnimated source code is separate from Dying Light, Chrome Engine, Techland, and Mixamo. Those names and assets remain the property of their respective owners.

This repository currently bundles a small target skeleton/template reference set and an example RPack so the validated export pipeline can be reproduced. Before publishing or redistributing the repository, confirm that you have permission to distribute:

- files under `reference/`;
- `common_anims_sp_pc.rpack` and its manifest;
- any screenshots, FBXs, extracted models, animations, or game data added later.

User-imported FBX files and generated build output are ignored by default and should not be committed unless their redistribution rights are known.
